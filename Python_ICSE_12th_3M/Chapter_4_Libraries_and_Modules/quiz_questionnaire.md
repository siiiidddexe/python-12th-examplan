# Quiz: Libraries and Modules

**Instructions**: Answer the following questions to test your understanding of Chapter 4.

## Section A: Multiple Choice Questions (1 Mark each)

1.  Which statement correctly imports the `math` module?
    a) `include math`
    b) `import math`
    c) `using math`
    d) `require math`

2.  Which function generates a random integer between 1 and 10?
    a) `random.random(1, 10)`
    b) `random.randint(1, 10)`
    c) `random.int(1, 10)`
    d) `random.choice(1, 10)`

3.  What does `math.ceil(4.3)` return?
    a) 4
    b) 4.3
    c) 5
    d) 5.0

4.  Which module is used to work with dates and times?
    a) `time`
    b) `date`
    c) `datetime`
    d) `calendar`

5.  What is the value of `__name__` when a module is run directly?
    a) The module's filename
    b) `"__main__"`
    c) `None`
    d) `"__module__"`

## Section B: Short Answer Questions (2 Marks each)

6.  What is a module in Python? Why are modules useful?
7.  Differentiate between `import math` and `from math import sqrt`.
8.  What does the `dir()` function do?
9.  Explain the purpose of the `__name__` variable in a module.
10. Name any two functions from the `random` module and explain their use.

## Section C: Programming Questions (5 Marks each)

11. Write a program that uses the `math` module to calculate the area of a circle given its radius (use `math.pi`).
12. Create a user-defined module named `calculator.py` with functions `add()`, `subtract()`, `multiply()`, and `divide()`. Import and use this module in another file to perform operations on two numbers.
13. Write a program using the `random` module to simulate rolling a dice 10 times and print the results.
14. Create a module `student.py` that contains a class `Student` with attributes `name` and `marks`. Add a method to display student details. Import this module in another file and create 3 student objects.

1. b
2. b
3. c
4. c
