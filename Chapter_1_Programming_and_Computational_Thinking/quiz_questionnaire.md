# Quiz: Programming and Computational Thinking

**Instructions**: Answer the following questions to test your understanding of Chapter 1.

## Section A: Multiple Choice Questions (1 Mark each)

1.  Which of the following is NOT a valid variable name in Python?
    a) `_myVar`
    b) `myVar2`
    c) `2myVar`
    d) `my_var`

2.  What is the output of `print(10 // 3)`?
    a) 3.33
    b) 3
    c) 3.0
    d) 1

3.  Which data type is immutable?
    a) List
    b) Dictionary
    c) Set
    d) Tuple

4.  What keyword is used to define a function in Python?
    a) `func`
    b) `define`
    c) `def`
    d) `function`

5.  What is the result of `True and False`?
    a) True
    b) False
    c) Error
    d) None

## Section B: Short Answer Questions (2 Marks each)

6.  Differentiate between `/` and `//` operators with an example.
7.  Explain the difference between a List and a Tuple.
8.  What is the purpose of the `range()` function in a for loop? Give an example.
9.  Define Recursion. What is a base case?
10. Write a Python statement to take an integer input from the user.

## Section C: Programming Questions (5 Marks each)

11. Write a program to check if a number entered by the user is Prime or Composite.
12. Write a function `is_palindrome(s)` that takes a string `s` and returns `True` if it is a palindrome, and `False` otherwise.
13. Using a loop, write a program to print the Fibonacci series up to `n` terms.
14. Create a dictionary to store the names and marks of 5 students. Write a script to print the name of the student with the highest marks.

---
**Answer Key (Section A)**
1. c
2. b
3. d
4. c
5. b
