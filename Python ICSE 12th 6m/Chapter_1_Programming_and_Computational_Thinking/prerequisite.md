# Day 0 1-2 Hrs Prerequisites for Chapter 1: Programming and Computational Thinking

To successfully undertake this chapter, students should have:

1.  **Basic Computer Literacy**: Familiarity with using a computer, managing files, and using a text editor or an IDE (Integrated Development Environment).
2.  **Mathematical Logic**: A basic understanding of arithmetic operations, boolean logic (True/False), and logical reasoning.
3.  **Previous Knowledge (Class 11 Syllabus)**:
    *   Understanding of what a programming language is.
    *   Basic algorithmic thinking (Flowcharts and Pseudocode).
    *   Familiarity with the concept of variables and data representation.

## Recommended Tools
*   **Python Interpreter**: Python 3.x installed.
*   **IDE/Text Editor**: IDLE (standard with Python), VS Code, or PyCharm.
